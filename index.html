Battleship

Users will be able to play battleship online against a computer AI. Battleship is a game of strategy and luck. Users will place their ships in the ocean on a spacial grid representing coordinates of the ships. Once all the ships have been placed the user will then choose a coordinate to launch missiles at. If the user hits the opponates ship the square will be red, if not it will be yellow. A basic history of the game battleship can be found here. Software Design

Game

Player wil be facing a computer for the game of battle ship, the computers ships will be randomly placed, the user will be able to choose where they want their ships place

Ships Object

  • shipID: Unique id for ship
  • type: ship type. Example: Carrier
  • size: How big the ship is
  • orientation: Vertical or Horizontal, Position the ship is oriented.
  • hits: How many times the ship has been hit
  • maxHits: Amount of hits before the ship is sunk

tableStates object

  • blanks: 0
  • hit: 6
  • miss: 7

Table Array

2D array with columns and rows for the table

Functions/Methods

createTableArray(height, width)

Table constructor that creates a table array setting all values to 0

placeShip(shipName, orientation, coordinates)

Sets the ship in the table array based off of id and coordinates

checkIfHit(coordinates)

Checks to see if the coordnates contain a ship and calls the ship hit func or sets the square to missed state

hitShip(coordinates)

Finds ship in array of ships, increments hits on object, changes square to hit state

UpdateTable()

Updates the table to the table array after events happen. (hit, miss)
