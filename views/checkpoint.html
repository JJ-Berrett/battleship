Generate Player Grid

Generates a player grid based on size

Launching Missiles

Player can click grid to launch a missile to see if that square has a ship
For the sake of the current state the ships are currently displayed. However if clicked will be marked red, ocean squares will be grey and once a ship is sunk a notification will appear

Checking if a ship is sunk

Ship object that has an id, name, maxhits, and hits. Once the ship has been hit it will increment the hits number.
If all the ships squares have been hit then a notification will appear.
